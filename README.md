## ğŸ¨ Customization

### Changing Colors
The theme uses CSS custom properties (variables) defined in `:root`. To customize colors:

```css
:root {
    --bg-primary: #0D1117;       /* Main background */
    --accent-primary: #3B82F6;   /* Primary accent (blue) */
    --accent-secondary: #14B8A6; /* Secondary accent (teal) */
    --text-primary: #F8FAFC;     /* Main text color */
}
```

### Adding New Sections
1. Add HTML structure in `index.html`
2. Style with CSS in `style.css`
3. Add any interactivity in `script.js`
4. Update navigation links

### Updating Content
- Personal information: Edit text in `index.html`
- Skills: Modify the skills grid in the Skills section
- Projects: Update project cards with new information
- Contact details: Change contact information as needed
